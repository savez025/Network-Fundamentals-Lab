# Network-Fundamentals-Lab
Learning networking concepts for cybersecurity including IP addresses, DNS, protocols, ports, and network troubleshooting.

This repository documents my learning journey of networking concepts used in cybersecurity.

## Topics Covered

### IP Addressing

* Public IP
* Private IP
* IPv4 Basics

### DNS

* DNS Records
* Domain Resolution

### Protocols

* HTTP
* HTTPS
* TCP
* UDP

### Common Ports

* 22 (SSH)
* 25 (SMTP)
* 53 (DNS)
* 80 (HTTP)
* 443 (HTTPS)

## My Learning Goal

To build a strong foundation in networking concepts required for cybersecurity.
